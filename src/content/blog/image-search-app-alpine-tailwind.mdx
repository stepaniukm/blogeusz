Alpine.js is a lightweight JavaScript framework that lets you add reactivity to your HTML without the overhead of a full framework like React or Vue. Combined with Tailwind CSS, it's a great stack for building small, interactive components.

## What we're building

A minimal image search app that:
1. Takes a search query from the user
2. Fetches results from the Unsplash API
3. Displays them in a responsive masonry grid

## The markup

```html
<div x-data="imageSearch()" class="p-4">
  <input
    x-model="query"
    @keyup.enter="search()"
    class="w-full rounded border px-4 py-2"
    placeholder="Search images..."
  />
  <div class="mt-4 columns-3 gap-4">
    <template x-for="image in results" :key="image.id">
      <img
        :src="image.urls.small"
        :alt="image.alt_description"
        class="mb-4 rounded"
      />
    </template>
  </div>
</div>
```

## The Alpine component

```javascript
function imageSearch() {
  return {
    query: '',
    results: [],
    async search() {
      const res = await fetch(
        `https://api.unsplash.com/search/photos?query=${this.query}&client_id=YOUR_KEY`
      );
      const data = await res.json();
      this.results = data.results;
    }
  };
}
```

Simple, fast, zero build step. That's the Alpine way.
